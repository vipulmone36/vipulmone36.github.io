TREX (TRee EXploration)

In this project, I developed a system for guiding users in virtual object manipulation, focusing on precise movement control.

This project highlights my expertise in human-computer interaction and my commitment to advancing intuitive technologies for practical engineering design and assembly applications.

Inspired by the space colonization algorithm, I crafted a solution to help users navigate from a source to a target location, avoiding obstacles. This was done by integrating visual and haptic feedback to enhance user interaction and enable smoother virtual assembly tasks.
